# O QUE SÃO DATAFRAMES?

SÃO ESTRUTURAS DE DADOS FUNDAMENTAIS NO ECOSSISTEMA DO PANDAS, É UMA ESTRUTURA TABULAR BIDIMENSIONAL, SIMILAR A UMA PLANILHA OU TABELA DE BANCO DE DADOS.

import pandas as pd

# Criando um DataFrame simples
dados = { 'Nome': ['Mirla', 'Ana', 'Daniel'],
   'Idade': [25, 30, 35],
              'Salario': [5000, 6000, 8000]}
df = pd.DataFrame(dados)
df

# O QUE SÃO SERIES?

UM SERIES POR SUA VEZ É UMA ESTRUTURA UNIDIMENSIONAL QUE ARMAZENA UMA SEQUENCIA DE VALORES INDEXADOS.

PODEMOS DIZER QUE UMM DATAFRAME É NADA MAIS QUE UMA SEQUENCIA DE VÁRIOS SERIES NOMEADOS E ORGANIZADOS POR COLUNAS E ORDENADOS PELOS INDICES.


#O QUE SÃO OUTLIERS?

OUTLIERS SÃO VALORES "FORA DA CURVA"

PONTOS DISTANTES DO PADRÃO

Por que isso é MUITO importante?

Porque outliers:

    Distorcem a média
    Podem indicar erro de dado
    Podem indicar casos especiais muito valiosos
    Podem quebrar modelos de Machine Learning
    Contam histórias escondidas no dado
