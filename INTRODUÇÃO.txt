# CURSO ALURA DATA SCIENCE

NESSE CURSO VOU ABORDAR A ANÁLISE EXPLORATÓRIA DE DADOS UTILIZANDO DIVERSAS FERRAMENTAS FUNDAMENTAIS 
QUE FORMAM A BASE ESSENCIAL DO PYTHON PARA O TRABALHO COM DADOS

UTILIZANDO AS BIBLIOTECAS: PYTOH, SEABORN, MATPLOTLIB E NUMPY

Analisar dados não é um curso teórico, mas também não é um curso 100% prático. 
Nos valeremos de muita prática, carregando um conjunto de dados real com notas e avaliações de filmes para explorar.

Neste curso, vamos analisar e explorar dados relativos a filmes e notas dos sites do MovieLens e TMDB (The Movie Database)

Utilizaremos uma ferramenta da Google chamada Colaboratory, que é similar ao Jupyter Notebook.
